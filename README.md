fake-news-detection
 
## Project Overview

TruthChain is a system designed to detect fake news using Artificial Intelligence and store verification results on blockchain to ensure transparency and prevent tampering.
The system allows users to submit a news article or URL, analyze it using a machine learning model, and receive a prediction indicating whether the news is real or fake. The verification result is then securely stored on blockchain.
## Problem Statement

With the spread of misinformation on the internet at an alarming rate, it has become challenging to identify reliable news sources on the internet. Misinformation caused by fake news results in confusion, panic, and even manipulation of public opinion.
The objective of this project is to create a system that is able to detect fake news and keep records of verification results securely and transparently.
## Objectives

Detect fake news using machine learning techniques.
Provide transparency by storing verification results on blockchain.
Visualize fake news trends using data analytics.
Build a scalable backend system using enterprise system architecture.
## System Architecture

The system consists of the following components:
User → Frontend → Backend API → AI Model → Blockchain Storage → Analytics Dashboard
